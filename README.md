# Music Connects Bot

Telegram-бот для поиска людей со схожими музыкальными вкусами.

## Возможности

- Регистрация с анкетой (имя, группа, любимые исполнители)
- Поиск пользователей по имени исполнителя
- Отправка запросов на знакомство
- Принятие/отклонение запросов
- Список своих знакомств
- Редактирование профиля

## Стек

Python 3.11 · aiogram 3 · SQLAlchemy 2 · SQLite · Docker
